# Tasha Event Optimizer

A web-based calculator to help players optimize Stellarite spending during Tasha events.

## Features

- Multi-phase optimization logic
- Hybrid calculation path
- EN / VI language support
- Pack integration
- PP overflow warning
- Open-source (MIT)

## Live Version

[Visit Calculator](https://muffintashacalc.netlify.app/)

## How It Works

The calculator evaluates:
- Current resources
- Available Stellarites
- Pack purchases
- Multi-phase optimal allocation

It determines the most efficient path to maximize total Tasha.
